hexo搭配github pages搭建個人部落格(二)

Github設定

註冊好github後新增一個Blog的專案，Repository name是 你的帳號名稱.github.io
例如我的就是 anyuexue77.github.io ，剩下的不用動，填完後點選Create repository

我已經有一樣的名稱了所以會報錯

Git配置

再隨意地方打開CLI介面，輸入以下指令設定使用者名稱和電子信箱

git config --global user.name "Github使用者名稱"
git config --global user.email "電子信箱"

完成後可輸入以下指令查看是否設定完成

git config --global user.name
git config --global user.email

SSH配置

生成金鑰

ssh-keygen -t rsa -C "電子信箱"

生成的金鑰在C:\Users\電腦使用者名稱\.ssh下的id_rsa.pub

用記事本打開

複製後到Github右上角頭像setting裡找到SSH and GPG key

在SSH keys的地方選擇New SSH key把剛剛複製的key貼上點擊Add SSH key

完成後再CLI介面輸入以下指令出現successfully代表成功

 ssh -T git@github.com

hexo config設定

在部落格的資料夾裡找到_config.yml用記事本打開

找到deploy輸入以下內容

  type: git
  repository: git@github.com:傳案位置
  #例如git@github.com:anyuexue77/anyuexue77.github.io.git
  branch: master

*注意冒號後面要空一格

剩下設定可以到Hexo的官方文件查看說明

上傳至github

在CLI介面輸入以下指令

npm install hexo-deployer-git --save

之後就可以在bolg的資料夾裡面輸入hexo g後再輸入hexo s:啟動本利伺服器

查看bolg是否正常

*注意每次修改玩bolg後都要執行hexo g來生成靜態頁面，在上傳至github不然bolg不會有更改

在瀏覽器網址列輸入http://localhost:4000/或是127.0.0.1:4000來查看本地的bolg

沒問題後按Ctrl+C來停止伺服器，使用hexo d來部屬至github

做到這邊歌為可能會發現頁面跟我的不一樣，下一章會說怎模更換主題

常用指令

hexo s:啟動本地伺服器

hexo g:生成靜態頁面

hexo d:上傳至Github

hexo cl:清除靜態頁面
